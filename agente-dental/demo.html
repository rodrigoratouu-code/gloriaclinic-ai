⚙️ Primera vez usando el agente?
  1. Abre config.js y pega tu API key de Anthropic en API_KEY.
  2. Rellena los datos de tu clínica (teléfono, WhatsApp, email, etc.).
  3. Recarga esta página y haz clic en la burbuja azul abajo a la derecha.
  4. ⚠️ Producción: no dejes la API key en el frontend. Usa un backend proxy (Vercel Function, Cloudflare Worker, etc.) para proteger la clave.
